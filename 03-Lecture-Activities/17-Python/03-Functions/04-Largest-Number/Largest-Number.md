# Find The Largest Number

**Objective:** Write a function `find_largest(numbers)` that takes a list of numbers and returns the largest number.

**Example:**
```python
find_largest([10, 45, 2, 99, 6])  # Output: 99
find_largest([99, 45, 2, 10, 6])  # Output: 99
find_largest([10, 45, 2, 99, 6, 99])  # Output: 99
find_largest([10, 45, 2, 99, 6, 99, 100])  # Output: 100
find_largest([10, 45, 2, 99, 6, 99, 100, 100])  # Output: 100
find_largest([10, 45, 2, 99, 6, 99, 100, 100, 100])  # Output: 100
find_largest([10, 45, 2, 99, 6, 99, 100, 100, 100, 100])  # Output: 100
find_largest([10, 45, 2, 99, 6, 99, 100, 100, 100, 100, 100])  # Output: 100
find_largest([10, 45, 2, 99, 6, 99, 100, 100, 100, 100, 100, 100])  # Output: 100
find_largest([10, 45, 2, 99, 6, 99, 100, 100, 100, 100, 100, 100, 100])  # Output: 100
find_largest([10, 45, 2, 99, 6, 99, 100, 100, 100, 100, 100, 100, 100, 100])  # Output: 100
find_largest([10, 45, 2, 99, 6, 99, 100, 100, 100, 100, 100, 100, 100, 100, 100])  # Output: 100
find_largest([10, 45, 2, 99, 6, 99, 100, 100, 100, 100, 100, 100, 100, 100, 100, 100])  # Output: 100
find_largest([10, 45, 2, 99, 6, 99])  # Output: 99
```
